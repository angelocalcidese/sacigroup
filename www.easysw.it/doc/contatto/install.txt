Cg_form_to_mail script per inviare mail tramite form ad un indirizzo predefinito
   Copyright (C) 2005  Claudio Garau www.claudiogarau.it

########################################################

REQUISTI:
PHP

########################################################

Se lo script è stato regolarmente scaricato dovreste avere a vostra disposizione 5 file:
- "cg_form_to_mail.htm", form per l'iserimento dei dati
- "cg_form_to_mail.php", spedisce i dati all'indirizzo indicato
- "cg_form_to_mail.css", formatta il modulo in "cg_form_to_mail.htm"
- "install.txt", questo file :-)
- "license.txt", licenza GNU/GPL 

########################################################

INSTALLAZIONE:
Decomprimete il file .zip.
Modificate il "CSS" per adattarlo alla grafica del vostro sito.
Modificate l'indirizzo del destinatario nel file "cg_form_to_mail.php".
Includete il codice del file "cg_form_to_mail.htm" nel punto in cui volete che esso venga visualizzato.

Fatto!
