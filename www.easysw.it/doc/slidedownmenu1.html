Un bellissimo menù a slide -down con grafica esplicativa. Il menù si compone di un file html slidedownmenu1.html che richiama un file css slidedown-menu2.css per tutte le formattazioni e la composizione e colore dei caratteri e un file JavaScript slidedown-menu2.JS contenente lo Script Java. Il menù è componibile a piacere con Voci di Menù e di sottomenù.
Agendo sui vari parametri si possono variare dimensioni e colore delle voci di menù.

Davvero un bel menù da utilizzare per il vostro web
